# Посмотри в окно

Учебный проект по вёрстке интерфейса видеосервиса в рамках курса Яндекс Практикума.

## 📌 О проекте

Статическая страница с видеоплеером и формой поиска.  
Основной акцент сделан на аккуратной работе с Flexbox, состояниями интерактивных элементов и кастомной стилизации форм.

## ⚙️ Реализовано

- Семантическая HTML-разметка  
- Flexbox-вёрстка  
- Кастомные чекбоксы  
- Стилизация `:hover` и `:focus-visible`  
- Ограничение текста с использованием `text-overflow` и `line-clamp`  
- Кастомный скроллбар  

## 🛠 Технологии

- HTML5  
- CSS3
